# Agda-LLS
An Implementation of Various Linear Logics in Agda

Design
------

The design of Agda-LLS is based the following:
  - Haskell Frontend:
     - Parsers
       - Alex
       - Parsec
     - Repl
       - Haskeline
  - Agda Backend:
     - Type Checker
     - Evaluator
     - Pretty Printer
